Os primeiros computadores - Mark 1 e Colossus

(Voltar)

Vamos viajar no tempo para a era da Segunda Guerra Mundial, um período de intensa inovação e desenvolvimento tecnológico. Em meio ao conflito global, duas potências, os Estados Unidos e o Reino Unido, estavam secretamente trabalhando em projetos revolucionários que mudariam para sempre o curso da história da computação. Nos laboratórios da Universidade Harvard, nos Estados Unidos, o brilhante professor Howard Aiken liderava uma equipe na construção do Mark 1, um computador eletromecânico pioneiro.

Enquanto isso, do outro lado do Atlântico, na Grã-Bretanha, uma equipe de cientistas e engenheiros estava ocupada com a criação do Colossus, um computador eletrônico projetado para decifrar os códigos secretos utilizados pela Alemanha nazista. Ambos os projetos representavam avanços extraordinários na capacidade de processamento de dados e cálculos complexos. Enquanto o Mark 1 buscava automatizar tarefas matemáticas e científicas, o Colossus tinha a missão crucial de quebrar códigos inimigos para ajudar os Aliados na guerra.

Esses primeiros passos na era da computação marcaram o início de uma jornada incrível rumo à revolução digital que moldaria o mundo moderno. O Mark 1 e o Colossus não eram apenas máquinas; eram símbolos de progresso, determinação e cooperação em um momento crítico da história.

Client x Server – Comunicação na Internet

(Voltar)

A comunicação na internet é baseada em um modelo chamado de client/server. Sendo que o cliente é o dispositivo que está consumindo os serviços disponíveis na rede. Na outra ponta está o servidor, que é o dispositivo que hospeda esses serviços.

Dizer que um site está hospedado significa dizer que ele está em um servidor, conectado 24/7 na internet. Ao apertar “enter” depois de digitar uma busca ou um endereço no navegador, essa ação se transforma em uma requisição que trafega na rede até achar o servidor que atende essa solicitação. Chegando lá, ele devolve esse pedido do jeitinho que foi requisitado.

Não é responsabilidade do servidor:

Você acessou um site pelo PC, tudo ok com ele. Em outro momento, acessa pelo celular, mas percebe que a página não se comporta da mesma maneira: elementos somem, ficam comprimidos ou embaralhados. A culpa não é do servidor. Essas responsabilidades estão do lado do cliente. Os desenvolvedores web devem ficar atentos a isso, aplicando técnicas de design responsivo ou mobile first, para que os usuários tenham uma experiência agradável, independente do dispositivo que utilizam. Espero ter ajudado, seja: adquirindo conhecimento, revisando temas ou matando uma curiosidade.

Como funciona o HTML?

(Voltar)

HTML, ou HyperText Markup Language, é a linguagem padrão para criação de páginas da web. Ele fornece a estrutura básica para o conteúdo de uma página, permitindo que os navegadores a exibam de forma adequada. O funcionamento do HTML é baseado em marcações, ou tags, que são utilizadas para identificar diferentes tipos de conteúdo, como títulos, parágrafos, imagens, links, entre outros elementos. Cada tag possui um propósito específico e contribui para a organização e apresentação visual da página.

Além das tags para estruturar o conteúdo, o HTML também permite a inclusão de atributos que fornecem informações adicionais sobre os elementos da página, como cores, tamanhos e links para outros recursos. No geral, o HTML funciona como uma linguagem de marcação que define a estrutura e o conteúdo básico de uma página web, sendo complementado pelo CSS (Cascading Style Sheets) para estilização e pelo JavaScript para interatividade e comportamento dinâmico. Juntos, esses três componentes formam a base do desenvolvimento web moderno.

O que são TAGS?

(Voltar)

As tags são elementos fundamentais do HTML, sendo utilizadas para marcar e identificar diferentes tipos de conteúdo em uma página da web. Elas são compostas por palavras-chave ou abreviações envoltas por colchetes angulares, como <tag>. As tags podem ter atributos que fornecem informações adicionais sobre o elemento marcado. As tags desempenham papéis específicos na estrutura e formatação de uma página web, permitindo que os navegadores interpretem e exibam o conteúdo de maneira apropriada.

Por exemplo, a tag <p> é usada para indicar parágrafos, a tag <img> para inserir imagens, a tag <a> para criar links, entre muitas outras. Ao utilizar as tags corretamente, os desenvolvedores web podem organizar o conteúdo da página e garantir que ele seja apresentado de forma coerente e compreensível para os usuários. As tags são a base do HTML e desempenham um papel crucial na criação de páginas da web funcionais e visualmente atraentes.
